### 1. **Clone & Setup** (2 minutes)
```bash
git clone https://github.com/ashish-tandon/open-policy-platform.git
cd open-policy-platform
chmod +x scripts/*.sh
```

### 2. **One-Command Deployment** (15 minutes)
```bash

```

- **API Backend**: http://localhost:8000
